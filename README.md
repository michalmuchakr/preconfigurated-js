<h3>
  Before application could be started run:
</h3>

<code>
  $ npm i
</code>
<br />

<h3>
  In case of running appliation in development mode:
</h3>

<code>
  npm start
</code>

<h3>
  Then navigate to: 
</h3>

<code>
  http://localhost:8080/
</code>

<h3>
  To prepare production build run command:
</h3>

<code>
  npm build
</code>
